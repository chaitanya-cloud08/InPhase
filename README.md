# 🌙 InPhase – A PCOD Symptom Tracker

**InPhase** is a beautifully minimal and user-centric web app designed to help women with PCOD (Polycystic Ovarian Disease) track their daily symptoms, understand their hormonal patterns, and receive personalized wellness tips — all in one clean, interactive interface.

---

## ✨ Features

- ✅ Daily symptom logging (mood, sleep, stress, cravings, flow, energy, etc.)
- 🌗 Cycle phase detection (Menstrual, Follicular, Ovulation, Luteal)
- 💡 Phase-based tips and personalized suggestions
- 📊 Weekly and monthly visualizations of patterns and changes
- 🎨 Minimalistic UI with pastel palette and smooth animations
- 📁 Data stored securely in `localStorage` (optional backend ready)
- 📱 Fully responsive and mobile-friendly

---

## 🔧 Tech Stack

- ⚛️ **React** – Functional components with hooks
- 🎨 **Tailwind CSS** – Fast, responsive, modern styling
- 🧩 **Framer Motion** – Smooth animations for UI flow
- 📊 **Recharts / Chart.js** – For visualizing symptom trends
- 🧠 **Custom logic** – For phase tracking and smart tip generation
